# Homework 5 - Jekyll

# Global UFO Sightings Analysis

This chart explores the patterns and trends of unidentified flying objects based on reported sightings. By analyzing the data through interactive visualizations, we can uncover how reporting trends have shifted over the decades and which "shapes" dominate the public consciousness.

---

## Distribution of UFO Shapes
This visualization categorizes the frequency of sightings based on the described shape of the craft.

<vegachart schema-url="{{ site.baseurl }}/assets/json/ufo_shapes.json" style="width: 100%"></vegachart>

## Write-up: Plot 1
* **Description:** This bar chart displays the count of sightings for various reported UFO shapes, such as `'light', 'triangle', and 'circle'`."

* **Design Choices (Encodings):** 
  * X-Axis: Encoded with the count field as Quantitative (Q) data to represent the total volume of sightings.
  * Y-Axis: Encoded with the shape field using Nominal (N) data. I sorted the data using Altair. EncodingSortField to order the bars descending by count, prioritizing the most frequent sightings.
  * Text Layer: I added a secondary text mark layer that encodes the count as text, aligned to the right of each bar, to provide precise values to the user without them needing to check the axis.


* **Design Choices (Colormaps):** I utilized a conditional colormap based on a mouseover selection `(hover_select)`.
  * Default State: The bars use a light blue (#aec7e8).
  * Hover State: When a user hovers over a bar, it transitions to a darker blue (#1f77b4).
    * This choice uses color as a feedback mechanism for interactivity rather than as a primary data encoding, keeping the visual clean while rewarding user exploration.


* **Transformations:**
  * Cleaning: I stripped whitespace from the shape column and filtered out trash values such as: `'nan', 'null', 'undefined'` to ensure only valid categories were plotted.
  * Aggregation: I used Pandas `value_counts()` to aggregate the raw sightings into a summary dataframe `(shape_counts)` containing the unique shapes and their respective totals.

## Write-up: Plot 2

* **Description:** This visualization is an interactive line chart showing the history of UFO reports over time. It provides a temporal context, showing a massive spike in reported sightings starting in the late 1990s.


* **Design Choices (Encodings):** 
  * X-Axis: Encoded with the year field as Quantitative (Q). 
  * Y-Axis: Encoded with the count of sightings as Quantitative (Q).
  * Opacity: I used a conditional encoding for opacity. When the `"brush" selection` is active, years outside the selection fade to 20% opacity, while the selected range remains at 100%.


* **Design Choices (Colormaps):** 
  * A single-hue blue was chosen for the line and points to maintain consistency with the first chart. A separate colormap was not necessary because time is a continuous variable, and the primary goal is to show a trend line rather than categorical differences.


* **Transformations:**
  * Aggregation & Calculation: Inside the `stats_label` layer, I used `transform_aggregate` to calculate the sum, min, and max values of the selected range.
  * Label Generation: I used `transform_calculate` to create a dynamic string that displays the year range and the total count of sightings for the user's current selection.
  * Date Processing: In Python, I converted the datetime strings to objects and extracted the year component to facilitate grouping.
